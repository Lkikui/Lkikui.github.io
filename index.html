LearnUP Phonics Board

Web application built with Node.js, Websockets, Express, Bootstrap, Interact.js, and MongoDB.

About This Project

Web application developed with the Code For Good volunteer development team for non-profit organization LearnUP. The LearnUP Phonics Board is designed to optimize LearnUp's one-on-one learning sessions by digitizing their phonics board to teach struggling students how to read. The phonics board includes user online status indicators as well as moving tiles with phonemes, which update in real-time to allow for live instruction. This project also features an admin dashboard for instructors.

For this project I handled front-end and server-side code to propose and implement bug fixes and new features such as the user online status indicator. I also contributed formatting changes to improve user experience as well as design and build the home page of the application.
